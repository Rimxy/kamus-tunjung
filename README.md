# Kamus Digital Bahasa Tunjung - Indonesia

Aplikasi berbasis web untuk menerjemahkan kosa kata dan kalimat antara Bahasa Indonesia dan Bahasa Tunjung.

## Persyaratan Sistem

- PHP 8.2 atau lebih baru
- MySQL / MariaDB
- Web Server (Apache/LiteSpeed/Nginx)

## Cara Instalasi (Localhost)

1. Clone repository ini ke folder `htdocs` (XAMPP) atau `www` (Laragon).
2. Buat database baru di MySQL dengan nama `kamus_tunjung`.
3. Import file `database/kamus_tunjung.sql` ke dalam database tersebut.
4. Buka `koneksi.php` dan sesuaikan username dan password database Anda.
5. Akses `http://localhost/nama-folder-proyek` di browser.
